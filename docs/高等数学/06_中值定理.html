    第六讲 中值定理

    xuxiaolong 高等数学

    中值定理：考证明题，10'（①4‘+②6’）

    十大定理（1-4为f(x)，5-9为f'(x)，10为∫abf(x)dx）

    1. 定理1—有界与最值定理
    2. 定理2—介值定理
    3. 定理3—平均值定理

    # 一、涉及函数的中值定理

    # 1、定理1（有界与最值定理）

    设f(x)在[a,b]上连续，则m≤f(x)≤M，其中m，M分别为f(x)在[a,b]上的最小值与最大值.

    # 2、定理2（介值定理）

    设f(x)在[a,b]上连续，当m≤μ≤M时，存在ξ∈(a,b),使得f(ξ)=μ.

    # 3、定理3（平均值定理/离散的的平均值定理）

    当a<x1<x2<···<xn<b时，在[x1，xn]内至少存在一点ξ，使f(ξ)=f(x1)+f(x2)+···+f(xn)n.(算数平均值)

    注：

    定理3是（离散的）平均值定理：f(ξ)=1n∑i=1nf(xi).(算数平均值)(n是个数)

    定理10是（连续的）平均值定理：f(ξ)=1b−a·∫abf(x)dx=f(x)―.(b−a是区间长度)

    # 4、定理4（零点定理）

    当f(a)·f(b)<0时，存在ξ∈（a,b）,使得f(ξ)=0.（端点异号，经过x轴，由介值定理推论）

    # 例题

    1、设f(x)在[a,b]上连续，当a≤x1≤x2≤···≤xn≤b时，证明存在ξ∈[x1,xn],使得f(ξ)=f(x1)+f(x2)+···+f(xn)n.

    分析：由最值定理m≤f(x)≤M，所以m≤f(x1)≤M，m≤f(x2)≤M，···，m≤f(xn)≤Mn·m≤f(x1)+f(x2)+···+f(xn)≤n·Mm≤f(x1)+f(x2)+···+f(xn)n≤Mm≤μ≤Mm≤f(ξ)≤M所以f(ξ)=f(x1)+f(x2)+···+f(xn)n

    2、设f(x)在[a,b]上连续，当a≤x1≤x2≤···≤xn≤b时，证明存在ξ∈[a,b],使得f(ξ)=f(x1)+f(x2)+···+f(xn)n.

    # 二、设计导数（微分）的中值定理

    # 5、定理5（费马定理）

    设f(x)满足在x0点处{①可导,②取极值,则f′(x0)=0.

    费马大定理：xn+yn=zn，(n>2)没有正整数解。

    证明：

    证明导数在某一点等于几，用定义。f′(x0)=limx−x0f(x)−f(x0)x−x0=0

    # 6、定理6（罗尔定理）

    设f(x)满足{①在[a,b]上连续,②在(a,b)内可导，则存在ξ∈（a,b）,使得f′(ξ)=0,③f(a)=f(b).

    # 7、定理7（拉格朗日中值定理）

    $设f(x)满足\left{ \begin{aligned} ① && 在[a,b]上连续, \ ② && 在(a,b)内可导 \ \end{aligned} \right. $

    则存在ξ∈（a,b）,使得f(b)−f(a)=f′(ξ)(b−a)

    或者写成f′(ξ)=f(b)−f(a)b−a

    # 8、定理8（柯西中值定理）

    设f(x),g(x)满足{①在[a,b]上连续,②在(a,b)内可导，则存在ξ∈（a,b）,③g′(x)≠0.

    使得f(b)−f(a)g(b)−g(a)=f′(ξ)g′(ξ)

    # 10、定理10（积分中值定理/连续的平均值定理）

    设f(x)在[a,b]上连续，证明存在ξ∈[a,b],使得∫abf(x)dx=f(ξ)(b−a).

    证明：

    ①最值原理由于f(x)在[a,b]上连续，则f(x)有最大值M和最小值m⇩m≤f(x)≤M⇩md(x)≤f(x)d(x)≤Md(x),d(x)>0⇩∫abmd(x)≤∫abf(x)dx≤∫abMd(x)⇩m(b−a)≤∫abf(x)dx≤M(b−a)⇩②介值定理m≤∫abf(x)dxb−a≤M⇩m≤μ≤M⇩存在ξ∈[a,b]，使f(ξ)=μ⇩∫abf(x)dxb−a⇨f(ξ)⇨∫abf(x)dx=f(ξ)(b−a)
